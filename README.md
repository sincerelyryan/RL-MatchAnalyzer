# RL-MatchAnalyzer
Simple C# Program that uses RocketStats to retrieve info and send discord webhook about stats in game.


All Things used
- **BakkesMod** - Allows us to be able to use plugins and many extra features in game
- **RocketStats** - Plugin that retrieves stats about game
- **Discord.Webhook** - A NuGet Package that allows us to send webhooks without anything too extra
- **Costura.Fody** Embed all files into one packed .exe
